Indulge in the historic streets of Rome, the romantic canals of Venice, the breathtaking Amalfi Coast, and the cultural masterpieces of Florence. Take in the sights in this photo gallery exhibition and see how easy it is to replicate with some MDX magic and tailwind classes.

Features images served using `next/image` component. The locally stored images are located in a folder with the following path: `/static/images/italy/[filename].jpg`

Since we are using mdx, we can create a simple responsive flexbox grid to display our images with a few tailwind css classes.
