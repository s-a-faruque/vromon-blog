Explore Iceland's dramatic landscapes, featuring cascading waterfalls, massive glaciers, black sand beaches, and otherworldly volcanic fields. Take in the sights in this photo gallery exhibition and see how easy it is to replicate with some MDX magic and tailwind classes.

Features images served using `next/image` component. The locally stored images are located in a folder with the following path: `/static/images/iceland/[filename].jpg`

Since we are using mdx, we can create a simple responsive flexbox grid to display our images with a few tailwind css classes.
